Our Research

Advancing the frontiers of Brain-Computer Interfaces, Machine Learning, and Signal Processing.

Current Research

Hybrid Deep Learning for Real-Time Motor Imagery BCI

Developing a novel hybrid CNN-Transformer architecture to improve decoding accuracy and reduce latency in motor imagery-based Brain-Computer Interfaces.

Adaptive EEG Signal Processing Pipeline

Creating an adaptive real-time signal processing framework for artifact removal and feature enhancement in non-stationary EEG data.

Multi-Modal BCI for Cognitive Enhancement

Integrating EEG and fNIRS data with machine learning to develop next-generation cognitive monitoring and enhancement systems.

Neuro-Inspired Efficient Machine Learning Models

Designing lightweight, brain-inspired neural networks optimized for edge devices in portable BCI applications.

Past Research

EEG-Based Mental Workload Assessment

Investigated real-time mental workload classification using traditional signal processing and early machine learning techniques.

Low-Cost Non-Invasive BCI Prototype

Developed an affordable hardware-software integrated BCI system for basic cursor control using consumer-grade EEG headsets.

Deep Learning for Seizure Detection

Explored convolutional neural networks for early detection of epileptic seizures from long-term EEG recordings.
